1. Basic Javascript
Javascript bertindak seperti statement seperti kita ingin merubah tampilan dari website menjadi dark mode.
Comment pada javascript = // dan /* */
Variable berperan untuk membuat value, disarankan menggunakan camelcase.
Membuat variable dengan const dan let.
Ketika mengganti variable setelah didefinisikan sebagai const maka akan menyebabkan typeerror.
Kita bisa menggunakan template literal (`) yang bisa kita gunakan seperti apa bila ingin console log yang ada single dan double quote didalamnya.
Programming languange pada umumnya bertindak secara synchronous.
False value pada js adalah false , 0, "", null, undefined.

const boolean = true;
