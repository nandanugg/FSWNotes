2. Looping
Loop adalah pengulangan, bisa menggunakan while maupun for. 
While loop akan terus run sampai kondisi tidak lagi benar. 
const impostor = [false, false, true, false];
let looptimes = 0;
while (looptimes < impostor.length) {
    looptimes++;
    const item = impostor[looptimes];
    if (item) {
        console.log(`Impostor is found, its user number ${looptimes + 1}`);
    }
}
Biasanya loop digunakan untuk mencari data dari array dan filter untuk input sesuatu seperti rating seller dari suatu ecommerce dan menampilkan input berupa string, karena jika menggunakan conversion integer lebih rumit.
Penggunaan for of digunakan untuk menampilkan item 1 by 1 karena kita pastinya tidak mungkin console log setiap problem.
Kita bisa save data ke variable dan menggunakan template literal (`)
const temperatures = [32, 33, 35, 32];
for (const temperature of temperatures) {
    console.log(`the temperature is ${temperature}`)
}

4. Arrow Function
Arrow function bukanlah function tetapi adalah value seperti string. 
Function dapat memiliki lebih dari 1 parameter.
Synchronous dan Asynchronous berbeda dari cara kerja. 
Jika synchronous menunggu code untuk di eksekusi sesuai order. Asynchronous tidak menunggu, akan mengeksekusi sesuai yang dibutuhkan.
Contohnya jika code terakhir akan dieksekusi setelah 3 detik maka ketika kita call function tersebut, kita akan mendapat undefined karena delay.
function getMessage() {
    setTimeout(() => {
        return "this is the message from getMessage()"
    }, 3000)
    console.log("inside getMessage()", "getMessage")
}
console.log("outside getMessage()", getMessage());
