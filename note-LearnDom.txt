--DOM-- 

Document Object Model 

yang mana DOM merupakan tampilan interface pemrogrman yang berbasis object 
Document merupakan object yang terdiri dari beberapan method dan property 

untuk mengimplentasikannya bisa membuat file js dan HTML 
yang mana file HTML harus terkoneksi dengan js 
dengan cara <script src="namafile.js"></script>

untuk mempermudah melihat perubahan dan perintah yang di eksekusi di console
buat file HTML yang di dalamnya berisi tag yang berisi id dan juga class 

penjelasan dan contoh kodingan dari DOM ada di file script.js
