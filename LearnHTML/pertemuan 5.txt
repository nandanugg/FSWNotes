Komponen website --> HTML, CSS, JS

HTML (HyperText Markup Language) --> struktur website
CSS (Cascading Sytle Sheet) --> pemberi style/tampilan, karakter, warna, variasi,dll. Memperindah website
JS (JavaScript) --> membuat website lebih interaktif. Soal aksi

HTML adalah bahasa markup untuk membuat web pages. bahasa markup adalah bahsa yg mengurung sebuah teks untuk ditampilkan dg cara tertentu

tag HTML
semacam kode yg dibungkus dengan karakter enclosing tag (<>)

--- tag ---
h1 = menampilkan header. biasanya untuk judul
h2 menampilkan header. ada h1,h2,h3,h4,h5,h6
p menampilkan paragraf
strong = bolet

ANATOMI/Struktur HTML
- Doctype --> menunjukkan tipe file. tradisi yg dilakukan sebelum menulis HTML
- cara cepat menuliskan struktur html:
  ketik tanda seru(!), klik enter

HEAD
- dalam struktur html ada antribut lang="en"
  menunjukkan struktur yg kita buat dalam struktur bahasa inggris
- chartset="utf-8"
  "utf-8" adlah teks encoding. chartset mengkonversi bahasa program ke bahasa manusia
- lang = bahasa, charset = memberitahu komputer, karakter2 itu harus diubah ke bahasa manusia
- <title> = judul web yg ditampilkan dalam tab
- 1 html untuk satu bahasa
- <meta> --> menunjukan seberapa lebar konter 
- content="width=device-width --> sesuai dengan lebar website

BODY
semua yg muncul dalam browser

agar tanda2 ini dapat ditampilkan di web
< --> &lt;
> --> &gt;
" --> &quot;
' --> &apos;
& --> &amo;

misal
<p>menampilkan<p></p>, maka
<p>menampilkan &lt;p&gt; </p>
