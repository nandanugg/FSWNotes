1. Html
Git flow adalah bagaimana cara git bekerja
Pada javascript berperan menambah fitur ketika click seperti tokopedia, ketika button clicked maka akan ada pop up yang ditampilkan
Html dapat memodifikasi kalimat pertama dan terakhir.
Apabila tidak ada img pada copy link address, maka img tidak akan muncul pada website kita.
Pada tag button disabled value adalah boolean, apabila tidak ditulis maka akan diaggap true.

<img src="kenneth-branagh-as-hercule-poirot-murder-on-the-orient-express-037.jpg" alt="">
<button disabled="true">ciao</button>

doctype html mendeskripsikan bahwa file ini adalah html, ini merupakan tradisi yang sudah mendarah daging.
Didalam tag <html> terdapat browser accesibility seperti lang, yang memungkinkan untuk membuka pada search engine based on ip address country.
Jadi sistem kerja search engine bakal membaca dari atas head yang didalamnya seperti meta charset kemudian reference css.
terdapat text encoding seperti biasanya utf-8 yang menyatakan beberapa struktur bahasa.
Shortcut untuk menulis tag html ketika di vscode -> ! dan tab, voila semua html structure bakal terisi.
Sebagai referensi kita bisa melihat bagaimana struktur html di website lain dengan inspect element
more references: w3schools.com/html

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Hercule Poirot's</title>
</head>
