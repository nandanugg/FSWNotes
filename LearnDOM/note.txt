2. DOM
Pada javascript click listener merupakan fitur penting.
getElementsByClassName akan memanggil collection berdasarkan class, yang berarti array. Jadi ketika ingin mendapatkan value pertama gunakan index seperti array[0].
Untuk mendapatkan value menggunakan input.value
Jika membuat script html, declaration harus ditempatkan pertama kali kemudian logic dan parameter.
Kita juga bisa mendapatkan dan menset value seperti:
button onclick = () => {
   console.log('button pressed', input.value);
   input.value =""
} 
Apabila kita tidak memasukkan index maka akan mendapatkan typeerror. Selalu ingat kalo getElementBy adalah array yang bisa selalu kita loop. Kita bisa looping misalnya apabila lebih dari 2 p.
Id akan merefer ke elemen dan apabila ada 2 elemen dengan 2 id yang sama, akan tetap merefer ke id pertama. Cara kerja Id lebih spesifik daripada class.
