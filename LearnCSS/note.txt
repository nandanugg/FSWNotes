2. CSS Styling
Margin = jarak seperti 12 px, margin ibaratnya seperti tanah dan jarak bisa lebih besar dari padding.
Margin dan padding biasanya dikombinasikan secara bersamaan. 
Secara default browser akan memberikan margin dan beberapa warna untuk mendefinisikan setiap margin, padding, border
Penginputan paddin secara default: top, right, bottom, left. Sesuai jarum jam
Kita bisa memberikan warna pada padding tetapi tidak bisa memberikan warna pada margin.
Border juga bisa diberikan warna dengan solid argument.
div adalah tag tanpa function yang berperan untuk membagi beberapa elemen di html tag di body
. -> di css berarti class. Id biasanya digunakan di javascript karena bisa mereferkan 1 id dalam 1 elemen. Kita bakalan lebih sering gunakan class
Shortcut untuk membuat div tag html yang berisi class didalamnya -> .footer, footer bakal jadi class di dalam tag div
Untuk selain div bisa gunakan seperti pada p tag -> p.footer-text
.footer -> enter to shortcut in vscode with <div>
p.footer-text -> for p
Untuk membuat tag a yang ada class didalam p tag -> p>a.footer-link
Untuk menggabungkan keduanya -> p>a.footer-link+p.footer-explanation
Keyword untuk shortcut lain cek pada shortcut emmet html
Pada input disarankan untuk memiliki padding untuk memberikan ruang styling pada input
pseudoclass untuk animation = hover, biasa digunakan di engineer life sehari hari
.paragraph::first-letter{
    font-size: 50pt;
}



3. Bootstrap
Untuk membuat website kita lebih neat, menggunakan bootstrap framework
Bootstrap adalah koleksi dari css dan kita bisa gunakan itu. 
Penggunaan dengan cdn memiliki kelemahan seperti tidak bisa mengakses ketika tidak punya koneksi internet.
Bagaimana menggunakan CSS? 
a. first download in getbootstrap.com
b. extract file from zip
c. copy bootstrap.css to file you want to 
d. then @import 'bootstrap.css' to see what happen
Kita akan sering menggunakan grid dengan column dan row
col di css adalah width yang bisa memiliki 2 kelas pada satu elemen
<div class="related col">
        <h4>Let's just play some easy</h4>
</div>
