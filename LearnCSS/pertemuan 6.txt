Cara2 penulisan CSS   
Internal Sylesheet ---> CSSlangsung ditulis di file HTML. not recomended karena code akan menajdi panjang
Inline Style ---> langusng diketik dengan <style>. not recomended. rekomended jika hanya unutuk mengecek

Struktur penulisan

CSS
h1{                                                             ----> selector
    color: blue;
    background-color: yellow;                         ----> atribut
    border: 1px solid black;
}

atribut dibuka dengan kurung kurawa ({)
atribut memiliki nama atribut dan value. color itu atribut dan blue adlah value.
jika menggunakan atribut lebih dari satu wajib menggunakan semi-colom (;)
jika atribut salah, browser akan menskip nya
jumlah atribtu sanagt banyak dan tidak semua browser memiliki atribut yg sama

website untuk mengecek atribut CSS : caniuse.com

atribut CSS harus sesuai dengan target market
aplikasi untuk petani atau anak2 desa atau orang2 menengah kebawah maka pilih atribut CSS yg mendukung perangkat mereka

class = untuk mereferinsikan ini untuk apa sih
class boleh pakai bahasa apa aja
