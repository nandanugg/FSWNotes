1. Apa itu NodeJS?

NodeJS atau node adalah sebuah runtime javascript yang fungsinya adalah untuk menjalankan kode javascript tanpa bantuan web browser, sehingga kita dapat membuat sebuah aplikasi seperti sistem backend, desktop, mobile, dan lain-lain dengan menggunakan javascript

Contoh cara menjalankan file javascript dengan node:

```bash
node script.js
```

Perintah di atas akan menyuruh node untuk menjalankan file javascript yang kita buat dan dengan nama script.js

Untuk membuat sebuah project node, pertama kita perlu menjalankan perintah berikut:

```bash
npm init
```

Perintah di atas akan menjadikan project direktori kita sebagai project dari nodejs

2. Framework Express

Express adalah sebuah framework dari javascript yang mempermudah kita untuk melakukan pembuatan sebuah web server dengan nodejs

Untuk instalasi express, bisa menggunakan perintah berikut:

```bash
npm install express
```

Contoh untuk menggunakan express adalah seperti berikut:

```bash
const express = require('express')
const app = express()
const port = 3000

app.get('/', (req, res) => {
    res.send('Hello World!')
})

app.listen(port, () => {
    console.log(`Example app listening at http://localhost:${port}`)
})
```

- pertama kita import dulu framework express dengan menggunakan fungsi require()
- setelah itu kita referensikan fungsi express ke variable app
- kemudian, kita buat variable port untuk menentukan port dari web server kita
- kita menggunakan method get untuk membuat sebuah route dan memberikan result berupa teks 'hello world'
- method listen digunakan untuk memberikan port yang sudah kita tentukan tadi agar express menggunakan port tersebut sebagai listen port yang akan melayani request client dari browser

Untuk menjalankan file di atas, kita bisa menggunakan perintah:

```bash
node index.js
```

3. Modules dan Serving Request pada Express

Kita bisa mendefinisikan banyak route pada express yang dapat kita akses pada web browser.

Pertama, kita buat direktori baru dengan nama route

Setelah itu di dalam folder route, buat file baru dengan nama rootRoute.js dan isikan seperti berikut

```bash
const express = require('express')
const app = express.Router()

app.get('/', (req, res) => {
    res.send('Halo dunia')
})

module.exports = app
```

Penulisannya sama saja seperti contoh penggunaan express sebelumnya, hanya saja kali ini kita tidak mereferensikan fungsi express secara langsung pada variable app, melainkan kita mereferensikan sebuah method bernama Router() yang gunanya adalah memberi tahu express bahwa kita akan menggunakan file ini sebagai router kita.

Kemudian ubah isi dari index.js seperti berikut:

```bash
const express = require('express')
const app = express()
const port = 3000

const rootRoute = require('./route/rootRoute')
app.use(rootRoute)

app.listen(port, () => {
    console.log(`Example app listening at http://localhost:${port}`)
})
```

Kita definisikan sebuah variable rootRoute yang mereferensikan sebuah file javascript rootRoute yang kita buat sebelumnya.

Setelah itu kita menggunakan method use() untuk menggunakan route yang sudah dibuat, kemudian kita jalankan kembali node nya dengan perintah berikut

```bash
node index.js
```
