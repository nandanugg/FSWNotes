hari ini kita belajar mengenai Nodejs.
Nodejs adalah sebuah runtime javascript, atau sederhananya
kita bisa melakukan pengembangan app di sisi server atau backend
menggunakan bahasa pemrograman javascript.

1. membuat direktori LearnNodeWebserver
2. npm init
    dengan perintah ini kita menginisialisasi
    direktori LearnNodeWebserver menjadi sebuah
    direktori project Nodejs kita.

    sama halnya perintah git ini ketika ingin
    menginisialisasi direktori kita menjadi sebuah
    repository.

3. menginstal framework Nodejs yaitu Expressjs
    dengan memasukkan perintah | npm install Expressjs

4. setelah langkah 1 - 3 sudah dijalankan
    maka akan muncul folder node_modules dan 2 file yaitu
    package.json dan package-lock.json

5. membuat file .gitignore
    di dalam file ini kita bisa memasukkan
    file apa saja yg tidak ingin kita masukkan ke git

kemudian kita membuat folder route untuk file rootRoute dan
views untuk source code kita dan membuat file index.js

dan dilakukan beberapa konfigurasi pada file index.js, rootRoute.js
